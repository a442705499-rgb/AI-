## Project Standard

Each project should include:

- Clear problem statement
- Minimal reproducible implementation
- Training or inference instructions
- Evaluation result or qualitative analysis
- Link back to the related paper note

The purpose of this folder is to show not only that I read papers, but that I can turn ideas into working code and measurable experiments.
